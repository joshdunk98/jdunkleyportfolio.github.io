Path Planning Visualization Under Development Currently working on a program to visualize various path planning algorithms, such as A* and wave front. The application currently works by accepting a "world" represented in a .dat file, recording user input for the starting and ending locations in the "world", and then computing the shortest path based on the specified path planning algorithm. I have currently implemented only A* and wave front. I plan to implement better visulizations in the future using PyGame and TKinter packages to allow the user to create a dynamic world.
